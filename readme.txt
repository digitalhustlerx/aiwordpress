Context-aware WordPress AI Agent that acts as a Site Co-pilot. Talk to your website to change styles and content live.

== Description ==

AIWP Copilot is an agentic AI widget that lives inside your WordPress frontend and backend. It understands the context of the page you are viewing and can execute commands to modify CSS and content in real-time.

Features:
- **Hanging Widget**: A persistent floating AI assistant.
- **Provider Agnostic**: Use OpenAI, Gemini, or any OpenAI-compatible API (OpenRouter, Groq, etc.).
- **Element Picker**: Click any element to target it specifically.
- **Live Preview**: See changes instantly before applying.
- **Safe Persistence**: Changes are saved per-page/post safely.

== Installation ==

1. Upload the `aiwp-copilot` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **AIWP Copilot** in the admin sidebar.
4. Enter your API Key and configure your preferred provider.

== Frequently Asked Questions ==

= Does it work with Elementor? =
Yes! It can target Elementor elements and apply custom CSS overrides.

= Can I use my own API key? =
Yes, the plugin is BYOK (Bring Your Own Key).

== Settings ==
Configure your API keys and provider selection under the AIWP Copilot menu in your WordPress dashboard.
